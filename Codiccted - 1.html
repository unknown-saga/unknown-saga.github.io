My First Language -> Python

The first language that I learnt was Python. It was developed by Guido van Rossoum. He is a Dutch programmer. Python was created by the hybridization of different high-level languages like C, C++ and JAVA. It is such a language where programs are short and to the point. It has no complexity of syntaxes. It is a lightly typed language where only the necessary things are typed. It is a great choice for those people who are new to programming. I personally recommend this language to them. A Little More Experience -> HTML, CSS, JavaScript

HyperText Markup Language is a WebDev language which is important for all the coders and progarammers to learn. It is used to create websites that are used by different people for different functions. And be it education or entertainment, foodie or goodie, no matter what! Websites serve the way we want! And to make them work better, HTML is used to update them.

JavaScript is not at all the same as Java, and it has a lot of difference. Java can be used independently while JavaScript requires HTML and DevTools for working. This is used to controol the functions of a webpage.
